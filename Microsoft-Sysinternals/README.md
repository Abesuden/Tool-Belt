## Getting Started

These are stand alone executable, which do not need to be installed, and were produced internally at Microsoft but are available for download by anyone. You can find them by clicking on the hammer tool link icon or download Sysinternals [here](https://download.sysinternals.com/files/SysinternalsSuite.zip)

```
Sysinternals
```

Recommend Tools:

  - *procexp64* - super version of task manager
  - *processhacker* - same but more powerful than `procexp64.exe`
    - shows processes, services, network connection/disconnection activities
    - can view memory content of different processes
    - useful for tracking down viruses
    - can be found [here](https://processhacker.sourceforge.io/)
  - *procmon64.exe* - shows logged actions of every process on the computer
  - *treesize* - shows disk spaced used in storage devices
    - can be found [here](https://www.jam-software.com/treesize_free)
  - *notmyfault64.exe* - crash the computer in variety of ways (for testing)
